<h3>Workflow for human and baggage detection</h3>

**Human**
* **Video Capture** - OpenCV
* **Detection** - Reinspect
* **Tracking** - HOG, HAAR, MHT

**Baggage**
* **Detection** - Reinspect, YOLO(Single-/ Multi - class)
